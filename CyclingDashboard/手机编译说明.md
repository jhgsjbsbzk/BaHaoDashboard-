# 手机编译 APK 说明

这个项目已经加入 GitHub Actions 云编译配置，你不用电脑也能生成 APK。

## 你要做什么

1. 手机打开 GitHub，注册或登录账号。
2. 新建一个仓库，例如 `BaHaoDashboard`。
3. 上传本项目压缩包里的所有文件。
4. 打开仓库的 `Actions`。
5. 找到 `Build Android Debug APK`。
6. 点 `Run workflow`。
7. 等几分钟。
8. 进入最新一次运行记录，下载 `BaHao-CyclingDashboard-debug-apk`。
9. 解压后安装里面的 `.apk`。

## 音乐控制怎么开启

安装 APK 后：

1. 打开应用。
2. 点音乐按钮。
3. 系统会跳到“通知使用权”设置。
4. 找到并开启 `八号音乐控制`。
5. 返回应用。
6. 先打开 QQ 音乐、网易云、酷狗等开始播放。
7. 再点仪表盘音乐按钮，就可以控制当前手机正在播放的音乐。

支持：

- 播放 / 暂停
- 上一首
- 下一首
- 音量加
- 音量减
- 显示当前歌曲名
- 显示歌手
- 显示进度

## 注意

如果没有开启 `八号音乐控制` 的通知使用权，Android 不允许本应用控制其他音乐 App。

## 蓝牙钥匙怎么用

这个项目已经加入蓝牙钥匙逻辑。安装同一个 APK 到两台手机：

1. 钥匙手机打开 `八号蓝牙钥匙`。
2. 点 `启动钥匙模式`。
3. 车机手机打开 `八号蓝牙钥匙`。
4. 点 `启动车机模式并打开仪表盘`。
5. 车机手机会进入仪表盘。
6. 钥匙靠近，仪表盘亮起。
7. 钥匙离开或蓝牙信号消失约 7 秒，仪表盘纯黑锁车。

说明：

- 距离是通过蓝牙 RSSI 信号强弱估算，不可能刚好等于 1.5 米。
- 不同手机蓝牙功率不同，实际距离可能在 1 到 3 米附近浮动。
- 如果太灵敏或太迟钝，需要改 `BluetoothKeyService.kt` 里的 `NEAR_RSSI`。
- 数值越大越近，例如 `-60` 更近才开机，`-75` 更远也会开机。
